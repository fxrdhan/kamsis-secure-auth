# ciscotalos/snort3 repository overview

### Snort3 Container

This is an interactive container for learning Snort3 with the Cisco Talos Snort3 training videos.

#### Features

- Snort3 Tutorial
- Rule Writing Labs
- Detection Strategies

#### Step by Step Guide

##### Download the Container

# random\_bytes

(PHP 7, PHP 8)

random\_bytes — Get cryptographically secure random bytes

### Description [¶](https://www.php.net/manual/en/function.random-bytes.php\#refsect1-function.random-bytes-description)

**random\_bytes**([int](https://www.php.net/manual/en/language.types.integer.php)`$length`): [string](https://www.php.net/manual/en/language.types.string.php)

Generates a string containing uniformly selected random bytes with the requested `length`.


As the returned bytes are selected completely randomly, the resulting string is likely
to contain unprintable characters or invalid UTF-8 sequences. It may be necessary to
encode it before transmission or display.


The randomness generated by this function is suitable for all applications, including
the generation of long-term secrets, such as encryption keys.


The sources of randomness in the order of priority are as follows:

- Linux: [» getrandom()](http://man7.org/linux/man-pages/man2/getrandom.2.html), /dev/urandom

- FreeBSD >= 12 (PHP >= 7.3): [» getrandom()](http://man7.org/linux/man-pages/man2/getrandom.2.html), /dev/urandom

- Windows (PHP >= 7.2): [» CNG-API](https://docs.microsoft.com/en-us/windows/desktop/SecCNG/cng-portal)


Windows: [» CryptGenRandom](https://msdn.microsoft.com/en-us/library/windows/desktop/aa379942(v=vs.85).aspx)

- macOS (PHP >= 8.2; >= 8.1.9; >= 8.0.22 if CCRandomGenerateBytes is available at compile time): CCRandomGenerateBytes()



macOS (PHP >= 8.1; >= 8.0.2): arc4random\_buf(), /dev/urandom

- NetBSD >= 7 (PHP >= 7.1; >= 7.0.1): arc4random\_buf(), /dev/urandom

- OpenBSD >= 5.5 (PHP >= 7.1; >= 7.0.1): arc4random\_buf(), /dev/urandom

- DragonflyBSD (PHP >= 8.1): [» getrandom()](http://man7.org/linux/man-pages/man2/getrandom.2.html), /dev/urandom

- Solaris (PHP >= 8.1): [» getrandom()](http://man7.org/linux/man-pages/man2/getrandom.2.html), /dev/urandom

- Any combination of operating system and PHP version not previously mentioned: /dev/urandom
- If none of the sources are available or they all fail to generate
randomness, then a [Random\\RandomException](https://www.php.net/manual/en/class.random-randomexception.php)
will be thrown.


> **Note**:
>
> Although this function was added to PHP in PHP 7.0, a
> [» userland implementation](https://github.com/paragonie/random_compat)
> is available for PHP 5.2 to 5.6, inclusive.

### Parameters [¶](https://www.php.net/manual/en/function.random-bytes.php\#refsect1-function.random-bytes-parameters)

`length`

The length of the random string that should be returned in bytes; must be `1` or greater.


### Return Values [¶](https://www.php.net/manual/en/function.random-bytes.php\#refsect1-function.random-bytes-returnvalues)

A string containing the requested number of cryptographically
secure random bytes.


### Errors/Exceptions [¶](https://www.php.net/manual/en/function.random-bytes.php\#refsect1-function.random-bytes-errors)

- If an appropriate source of randomness cannot be found,
a [Random\\RandomException](https://www.php.net/manual/en/class.random-randomexception.php) will be thrown.

- If the value of `length` is less than `1`,
a [ValueError](https://www.php.net/manual/en/class.valueerror.php) will be thrown.

### Examples [¶](https://www.php.net/manual/en/function.random-bytes.php\#refsect1-function.random-bytes-examples)

**Example #1 **random\_bytes()** example**

`<?php
$bytes = random_bytes(5);
var_dump(bin2hex($bytes));
?>`

Run code

The above example will output
something similar to:

```
string(10) "385e33f741"
```
